# Web Development Project 6 - *Weather Dashboard*

Submitted by: **Han Htoo Satt**

This web app: **collects data from WeatherbitAPI to present summary stats and detailed information of cities. A summary chart is displayed along with each item in the list that can be clicked to view more information about it**

Time spent: **4** hours spent in total

## Required Features

The following **required** functionality is completed:

- [x] **The app includes at least one unique chart developed using the fetched data that tell an interesting story**
- [x] **Clicking on an item in the list view displays more details about it**
  - [x] To ensure an accurate grade, your sidebar **must** be viewable when showing the details view in your recording.
